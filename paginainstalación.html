1. Pasos previos

Primero instalamos XAMPP haciendo clic en descargar.

Una vez instalado, abrimos el Panel de Control de XAMPP y activamos los módulos Apache, para que se active un servidor web local, y MySQL para iniciar el servidor de base de datos , dandole en Start.

Después, accedemos a phpMyAdmin desde http://localhost/phpmyadmin y creamos una nueva base de datos llamada dolibarr sin importar ninguna tabla ya que Dolibarr se encargará de crearlas.

Luego descomprimimos Dolibarr y accedemos a http://localhost/dolibarr/htdocs para iniciar la instalación.

2. Revisión de requisitos

Dolibarr revisa que nuestro sistema cumpla los requisitos mínimos de PHP. Si todo es correcto, se pulsa en Empezar.

Es necesario incluir la carpeta Dolibarr dentro del directorio htdocs de XAMPP.

3. Configuración del servidor web y base de datos

Se indican las rutas de instalación y datos de la base de datos:

  • Directorio web: C:/xampp/htdocs/dolibarr/htdocs
  • Documentos: C:/xampp/htdocs/dolibarr/documents
  • Servidor BD: localhost
  • Usuario: root
  • Base de datos: dolibarr

4. Finalización de instalación

Creo el usuario administrador para acceder al sistema. Ya se ha instalado Dolibarr.

Ahora se pueden añadir los módulos que se necesiten.
